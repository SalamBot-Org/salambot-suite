Comment l'utiliser

1. Intégrez le widget

Copiez le code ci-dessus et collez-le dans votre site web.

2. Testez la détection de langue

Essayez d'envoyer des messages dans différentes langues :

  • Français : "Bonjour, comment puis-je vous aider ?"
  • Arabe : "مرحبا كيف يمكنني مساعدتك؟"
  • Darija : "labas 3lik, kif nqdr n3awnk?"

3. Forcez une langue spécifique

Utilisez le code JavaScript suivant pour forcer une langue :

// Forcer le français window.salambot.languageStore.setLangOverride('fr'); // Forcer l'arabe window.salambot.languageStore.setLangOverride('ar'); // Forcer le darija window.salambot.languageStore.setLangOverride('ar-ma');
